## ğŸ† Project Summary

**VisaSight** is an end-to-end machine learning application that:

1. **Predicts visa application outcomes** (Approved, RFE, Denied)
2. **Estimates processing times** with confidence intervals
3. **Explains predictions** using SHAP feature importance
4. **Monitors policy changes** in real-time

### Tech Stack
- **Frontend**: Next.js 14, React 18, Framer Motion
- **Backend**: FastAPI, Python 3.10
- **Database**: Supabase (PostgreSQL)
- **ML**: XGBoost, LightGBM, Transformers
- **Deployment**: Vercel, Railway, GitHub Actions
